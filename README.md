# Introduction to Databases for Back-End Development

This repository contains my learning progress and practice related to databases and their role in back-end development.

## Modules Covered

### Module 1: Database Fundamentals
- What databases are and why they are used
- Types of databases
- Basic database concepts

### Module 2: Relational Databases
- Relational database model
- Tables, rows, and columns
- Primary and foreign keys
- Relationships between tables

### Module 3: SQL Fundamentals
- Writing basic SQL queries
- SELECT, INSERT, UPDATE, DELETE operations
- Filtering and sorting data

### Module 4: Database Design
- Database schema design
- Normalization concepts
- Avoiding data redundancy

### Module 5: Practical SQL
- Working with real datasets
- Joining tables
- Applying SQL in back-end scenarios

## Goal

To build a solid foundation in databases and confidently use SQL and relational database concepts in back-end development.
